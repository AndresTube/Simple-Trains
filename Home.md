## What is SimpleTrains?

SimpleTrains is a lightweight Minecraft plugin that allows players to create train station networks and fast-travel between them using minecarts. Perfect for survival servers, RPG worlds, or any server wanting an immersive rail transport system.

## Features Overview

### Station System
- Create stations on rails with a base block
- Link stations together for travel
- Public and private link types
- Station ownership and transfer

### Minecart Registration
- Register minecarts using renamed Name Tags
- Only registered minecarts can warp

### GUI Interface
- Easy-to-use inventory menus
- Manage stations without commands
- View and accept link requests

### Full Customization
- All messages editable via `messages.yml`
- Full multi-language support
- Customizable creation block

### Sound Effects (v1.3.0)
- Configurable departure and arrival sounds
- Adjustable volume and pitch
- Can be disabled entirely

### Creation Costs (v1.3.0)
- Optional cost to create stations
- Supports XP levels, Vault economy, or items
- Admins bypass all costs

### Travel Costs (v1.3.0)
- Optional cost per travel
- Supports XP levels, Vault economy, or items
- Admins bypass all costs

### Tab Completion (v1.3.0)
- Full autocomplete for all commands
- Smart suggestions based on context

## Requirements

- **Minecraft:** 1.20.x or higher
- **Server:** Paper, Spigot, or forks
- **Java:** 17 or higher

---

## Installation

1. Download `SimpleTrains-1.3.0.jar`
2. Place it in your server's `plugins` folder
3. (Optional) Install [Vault](https://www.spigotmc.org/resources/vault.34315/) for economy-based travel costs
4. Restart or reload your server
5. Configure `config.yml` and `messages.yml` as needed
